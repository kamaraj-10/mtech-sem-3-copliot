### From QP2 - Sentiment Analysis:

```
**Calculation Example:**

Review: "The food was amazing, but service was slow"

Aspect 1: Food + "amazing" = Positive (0.95)
Aspect 2: Service + "slow" = Negative (-0.87)

Overall = (0.95 + (-0.87)) / 2 = 0.04 ≈ Neutral/Slightly Positive
```

### From QP2 - LSTM Calculation:

```
**Step 1: Forget Gate**
f_t = sigmoid([h_{t-1}, x_t] * W_f) = 0.645

**Step 2: Input Gate**
i_t = sigmoid([h_{t-1}, x_t] * W_i) = 0.599

**Step 3: Cell State Update**
C_t = 0.645 * C_{t-1} + 0.599 * candidate_value
```
