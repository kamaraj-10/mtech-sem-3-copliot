## 📝 QUICK REFERENCE FORMULAS

### Softmax (Probability)
$$P(x_i) = \frac{e^{x_i}}{\sum_j e^{x_j}}$$

### Cross-Entropy Loss
$$L = -\sum_i y_i \log(p_i)$$

### Attention Score
$$\text{Attention}(Q, K, V) = \text{Softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

### LSTM Cell State Update
$$C_t = f_t \odot C_{t-1} + i_t \odot \tilde{C}_t$$

### TF-IDF
$$\text{TF-IDF}(t,d) = \text{TF}(t,d) \times \log\left(\frac{N}{\text{df}(t)}\right)$$

## ⚡ COMMON MISTAKES TO AVOID

1. ❌ Not preprocessing text properly
   ✅ Always clean, tokenize, and normalize first

2. ❌ Confusing Intent with Entity
   ✅ Intent = What user wants; Entity = Specific data

3. ❌ Ignoring dialogue context
   ✅ Always maintain conversation state

4. ❌ Skipping calculations
   ✅ Show all steps: Input → Process → Output

5. ❌ Not validating outputs
   ✅ Always verify: Do results make sense?

6. ❌ Using RNN for long sequences
   ✅ Use LSTM or Transformer instead

7. ❌ Forgetting normalization in softmax
   ✅ Always normalize scores before taking exp()

8. ❌ Not considering edge cases
   ✅ Test with: empty input, special characters, long text
